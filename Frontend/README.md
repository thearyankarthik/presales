\# Presales Project

This repository contains the Presales web application built using Angular.

---

## Project Overview

The Presales system is designed to manage dashboards, project registrations, and related workflows for a presales team.

---

## Development Setup

This project was generated using [Angular CLI](https://github.com/angular/angular-cli).
